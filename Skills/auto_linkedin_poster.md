# Auto LinkedIn Poster - Silver Tier Skill

## Overview
Automatically drafts LinkedIn posts from sales/business leads detected in `/Needs_Action`. Follows HITL (Human-In-The-Loop) process requiring approval before publishing.

## Trigger
Scan `/Needs_Action` folder for markdown files with:
- `type: email`, `type: whatsapp_message`, or `type: linkedin_notification`
- Keywords in content: `sales`, `client`, `project`

## Process Flow

```
1. Scan /Needs_Action for sales leads
2. Extract service/benefit from lead content
3. Draft LinkedIn post using Company Handbook tone (polite, professional)
4. Save draft to /Plans/linkedin_post_[date].md
5. Move draft to /Pending_Approval for HITL review
6. Wait for human approval before publishing
```

## Usage

### Command Line
```bash
python Skills/auto_linkedin_poster.py
```

### With PM2 (Continuous Monitoring)
```bash
pm2 start Skills/auto_linkedin_poster.py --name "linkedin-poster" --interpreter python
```

### Example Trigger
```
@Auto LinkedIn Poster process sales lead
```

## Dependencies

```bash
pip install pyyaml
```

## Configuration

- **Scan Interval:** 300 seconds (5 minutes)
- **Keywords:** sales, client, project, lead, opportunity
- **Output:** `/Plans/linkedin_post_[YYYY-MM-DD_HH-MM-SS].md`
- **Approval Queue:** `/Pending_Approval/`

## Post Template

Based on Company Handbook guidelines (polite, professional language):

```
Excited to offer [service] for [benefit]! DM for more.

#BusinessGrowth #ProfessionalServices #Opportunity
```

## File Structure

### Draft Format (/Plans/)
```markdown
---
type: linkedin_draft
source: [original lead file]
created: [ISO timestamp]
status: draft
keywords: [matched keywords]
---

# LinkedIn Post Draft

## Content

[Post content here]

## Source Lead

- **From:** [original sender]
- **Original Subject:** [subject]
- **Matched Keywords:** [keywords]

---
*Status: draft - Requires approval in /Pending_Approval/*
```

### Approval Format (/Pending_Approval/)
```markdown
---
type: linkedin_draft
source: [original lead file]
created: [ISO timestamp]
status: pending_approval
keywords: [matched keywords]
---

# LinkedIn Post Draft - PENDING APPROVAL

## Content

[Post content here]

## Action Required

- [ ] **Approve** - Move to /Approved/ and publish
- [ ] **Reject** - Move to /Rejected/
- [ ] **Edit** - Modify content and resubmit

---
*Generated by Auto LinkedIn Poster - Silver Tier*
```

## Rules (from Company Handbook)

1. **Always be polite** - Posts use professional, courteous language
2. **Flag for approval** - All posts require HITL approval before publishing
3. **No financial claims** - Avoid specific dollar amounts or guarantees
